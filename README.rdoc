= Baukis - 顧客管理システム

== 説明

Baukis は企業向けの顧客管理システム（Ruby on Rails 学習用サンプル）です。

== 推奨されるシステム環境

* CentOS 6.5 または Ubuntu 14.04
* Ruby 2.1.2
* MySQL 5.6 または PostgreSQL 9.3
